# AI Chatbot

## Features
- Private bot chat
- Public bot chat triggered by "-password" flag (default "robot ")
- get ip configuration with "ip" chat
- reboot system wuth "reboot" chat

## How to compile for Raspberry PI 4+
```
./build-pi.sh
# copy RPI-chatbot.tar.gz to Raspberry

# On Raspberry:
./install_chatbot.sh
```
